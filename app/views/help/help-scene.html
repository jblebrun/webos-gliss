Playing PreGlide
Tap on a box to select it. Tap on an empty space. If the box can slide along a path to the empty space, it will. The box you've currently will be highlighted to help you. Black squares are holes, you can not move boxes into holes.
If you get three or more boxes in a row, they will disappear.
Once you've placed all of the tiles for a level, you can continue to try to clear tiles. If you are unable to clear all of the tiles, click on "Done" to go to the next level.
If you clear 5 or more boxes (4 or more on harder levels) a shimmering wild tile will appear. this tile can be used to match any color. It can match multiple colors in one turn for example, the following configuration will clear all five tiles.
